5. Image creation 
5.1. Create a separate directory and a Dockerfile with the following content

1. Create an additional directory "public-html" within new directory 
2. Place an index.html file with your desired content in it 
3. Return to the directory where the Dockerfile is stored. Build an image: 
docker build -t my-docker-image .
